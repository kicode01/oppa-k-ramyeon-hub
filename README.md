# Oppa K-Ramyeon Hub

** Live Demo:** [oppakramyeonhub.great-site.net](http://oppakramyeonhub.great-site.net)

Welcome to the **Oppa K-Ramyeon Hub**! This is a complete e-commerce and point-of-sale web application designed for a K-Ramyeon store. It allows customers to build their own custom ramyeon bowls, earn loyalty points, and check out seamlessly.

## Features
* **Build-a-Bowl Cart System:** Interactive shopping cart with quantity grouping and live totals.
* **Noodle Passport:** A loyalty system where customers can earn and redeem points for free bowls.
* **Admin Dashboard:** A secure backend interface for staff to verify users, update passport points, and view detailed digital receipts of customer orders.
* **Modern UI:** Sleek, responsive design built with HTML, CSS, and Vanilla JavaScript.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** PHP
* **Database:** MySQL (PDO)

## Setup Instructions
1. Upload the files to your web hosting provider (e.g., InfinityFree) or local server (e.g., XAMPP).
2. Import the database `.sql` backup file into your MySQL database.
3. Open `api/db_connect.php` and update the database credentials (Host, Username, Password, Database Name) to match your server.
4. Open the website in your browser and enjoy!
